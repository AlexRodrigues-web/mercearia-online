# Mercearia

Bem-vindo ao **Mercearia**, um sistema de gerenciamento de mercearia completo, desenvolvido em PHP, com suporte a autenticação, gestão de produtos, usuários e muito mais.

---

## 📋 Funcionalidades Principais

- **Autenticação de Usuários**: Login seguro com controle de sessões.
- **Gestão de Produtos**: Cadastro, edição e exclusão de produtos.
- **Gestão de Usuários**: Controle de níveis de acesso e gestão de funcionários.
- **Dashboard**: Resumo de funcionários ativos, produtos em estoque e vendas totais.
- **Páginas Públicas e Privadas**: Suporte a páginas acessíveis apenas para usuários autenticados.
- **Sistema Responsivo**: Interface amigável e responsiva para dispositivos móveis e desktops.

---

## 🛠️ Tecnologias Utilizadas

- **Linguagem de Programação**: PHP 7+.
- **Banco de Dados**: MySQL/MariaDB.
- **Frontend**: Bootstrap, CSS personalizado.
- **Controle de Dependências**: Composer.
- **Log de Erros**: Monolog.
- **Servidor Local**: XAMPP ou equivalente.
